## Project Requirements
1. PHP 8
2. Node / NPM / Yarn

# Project Setup Steps
1. Run: composer install
2. Run: yarn
3. Run: yarn mix
4. Setup .env file and add twilio and db configuration 
5. Run: php artisan migrate
6. Run: php artisan db:seed
7. Run php artisan storage:link
8. Run: Go to /register and register first User 
It will automatically create make it **Super Admin**
